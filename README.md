# University Courses Management System

Лабораторная работа 1: Система управления курсами и преподавателями

## Описание

Система для управления учебными курсами и преподавателями в университете. Позволяет добавлять и удалять курсы, назначать преподавателей, записывать студентов и просматривать информацию о курсах.

**Основные возможности:**
- Управление онлайн и офлайн курсами
- Назначение преподавателей на курсы
- Запись студентов на курсы
- Просмотр курсов преподавателя и студентов курса

## Запуск проекта

### Требования
- .NET 8.0 SDK

### Локальный запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd UniversityManagement
```
Восстановите зависимости:

```bash
dotnet restore
```
Запустите тесты для проверки работы:

```bash
dotnet test
```
Проект готов к использованию. Основная бизнес-логика находится в классе CourseService, покрыта юнит-тестами.
